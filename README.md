# my_website

Personal website for Alexander David Brandt, hosted via GitHub Pages.

## Structure

- `index.html` — main page (CV / professional profile)
- `projects/` — standalone HTML projects, not linked from the main page

## Local editing workflow

1. Edit files in VS Code at `C:\dev\my_website\`
2. `git add .`
3. `git commit -m "description of change"`
4. `git push`

Changes to `index.html` go live at `https://alexanderbrandt256.github.io/my_website/` automatically after pushing.
